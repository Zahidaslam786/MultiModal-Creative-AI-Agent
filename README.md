A sophisticated AI system designed to bridge the gap between textual descriptions and visual intelligence. This project integrates **Image Synthesis** and **Visual Perception** into a single cohesive agentic workflow.

## 🚀 Overview
This agent functions as both a creator and an observer. It can synthesize high-fidelity images from complex prompts and then "see" and analyze those images to provide structured descriptive feedback. This demonstrates the power of multi-modal AI in creative and analytical pipelines.

## 🛠️ The Workflow (Step-by-Step)
1. **Latent Diffusion Synthesis**: The agent utilizes the **Stable Diffusion** pipeline (Text Encoder, UNet, and VAE) to generate high-quality images from text descriptions.
2. **Visual Perception & Analysis**: Using **Vision-Language Models** (like BLIP), the agent interprets the generated output, autonomously describing objects, styles, and artistic composition.
3. **Hardware & Inference Optimization**: The system is specifically optimized to run on **T4 GPUs** using `float16` precision and the `accelerate` library for high-speed, memory-efficient inference.
4. **End-to-End Pipeline**: A unified Python interface that handles the transition from text-to-image and image-to-text without manual intervention.

## 🔧 Technical Stack
- **Frameworks**: Hugging Face Diffusers & Transformers.
- **Core Library**: PyTorch.
- **Hardware Optimization**: Accelerate for efficient GPU memory management.
- **Modality**: Multi-modal (Text + Vision).

# 🤖 Multi-Modal Agentic AI: From Creative Art to Functional Travel Logic

An advanced AI ecosystem that integrates **Generative Art**, **Visual Perception**, **Autonomous Flight Booking**, and **Data Analytics**. This project demonstrates how a single agentic workflow can handle both creative and functional real-world tasks.

## 🚀 Project Modules & Workflows

### 1. ✈️ Module: Agentic Flight Assistant & SVG Generation
This module uses a structured knowledge base (Dictionary-based RAG) to process travel queries. Instead of just replying with text, the agent autonomously generates a visual **SVG Flight Ticket**.

**Key Techniques:** Intent Recognition, Tool-Calling, SVG Rendering.

### 2. 🎨 Module: Multi-Modal Creative Loop (Stable Diffusion + BLIP)
A seamless transition from creation to perception. The agent uses **Stable Diffusion v1.5** to synthesize images and **BLIP (Vision-Language Model)** to interpret them, creating an autonomous feedback loop.

**Key Techniques:** Latent Diffusion, Visual Captioning, Hardware Optimization (fp16).

### 3. 📊 Module: Technical LLM Expert & Data Analytics
This module transforms the agent into a Data Scientist. Using **Pandas** and specialized LLM personas, it performs deep data analysis and provides technical insights into complex datasets.

**Key Techniques:** Pandas Integration, Persona-based Prompting, Synthetic Data Analysis.

## 🛠️ Technical Stack
- **Orchestration**: Llama 3.2 (Ollama) & Agentic Workflows.
- **Vision**: Stable Diffusion v1.5, Salesforce BLIP.
- **Data**: Pandas, NumPy, JSON-based Knowledge Retrieval.
- **Interface**: SVG Rendering, Voice (STT/TTS) Integration.

## 🔧 Hardware Optimization
- Specifically tuned for **T4 GPUs**.
- Utilizes `float16` precision and the `accelerate` library for 2x faster inference and lower memory footprint.
